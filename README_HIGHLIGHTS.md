## 🗺️ OA Literature Roadmap Details

### Phase 1: Current ✅
- arXiv (primary)
- PMC (biomedical)
- 10,000 requests/day free

### Phase 2: Near-Term 🔄
- **bioRxiv** - Biology preprints
  - Focus: Developmental biology, ecology, evolution
  - Estimated: Q1-Q2 2024

- **medRxiv** - Medicine preprints
  - Focus: Public health, clinical research
  - Estimated: Q1-Q2 2024

### Phase 3: Extended 🔄
- **Other OA Repositories**
  - SSRN (Social sciences)
  - EarthArXiv (Geosciences)
  - PsyArXiv (Psychology)
  - ChemRxiv (Chemistry)
  - etc.

### Phase 4: Complete Ecosystem 🔄
- Unified search across all OA sources
- Cross-source bibliography linking
- Field-specific optimization
