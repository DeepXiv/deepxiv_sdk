### Installation / 安装

```bash
pip install deepxiv-sdk

# Full features / 完整功能
pip install deepxiv-sdk[all]
```

### First Command / 第一个命令

```bash
# English: This will auto-register a token
deepxiv search "agent memory" --limit 5

# 中文: 这将自动注册一个 token
deepxiv search "agent memory" --limit 5
```

### System Information / 系统信息

For troubleshooting and debugging information, run:

```bash
deepxiv debug
deepxiv debug --verbose
```

For API health check:

```bash
deepxiv health
```
